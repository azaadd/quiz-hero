1. What is the difference between map(), filter(), forEach() and find()?

Answer: map()

A map is a method that loops through each element or item of an array or array of objects and returns the value.


forEach() :

forEach is a method that loops through each element or item of an array or array of objects but not returns value.


filter() :

A filter is a method that loops through an array or array of objects and conditionally returns all the items it finds.


find() :

find is a method that loops through an array or array of objects and conditionally returns only the first element in that array.


2. What is the difference between local storage, session storage and cookies?

Answer: Local storage

When we enter any information or data in the local server, the local storage stores it. Later when we exit the server the local storage still stores our data or information and we can use it.


Session storage

When we enter any information or data in the local server, session storage also stores the data or information like local storage. But session storage no longer stores data when we exit the server. He deletes the data.


Cookies

A cookie is something that can store some information on the browser. For example, when we visit a site, cookies store the URL, search address, name, etc. of that site. And we can use them later if we want.
